🔍 O que cada parte faz? Avisa o navegador: "o que vem agora é HTML moderno". Sem isso o browser pode interpretar a página errado. A tag que envolve tudo. O lang="pt-BR" avisa que o idioma é português — ajuda o corretor ortográfico do navegador. Área de configurações que o usuário não vê na tela. É onde ficam o título da aba, o link pro CSS e outras instruções pro navegador. Define o "alfabeto" usado na página. Com UTF-8, acentos como ã, é, ç aparecem certinho. Faz a página caber no celular. Sem isso, o site aparece minúsculo no smartphone. Conecta o CSS ao HTML. É como dizer: "vai buscar o visual nesse arquivo aqui". O href é o nome do arquivo. Tudo que aparece na tela fica aqui. Header, textos, imagens, botão — é dentro do body.
O topo da página. Fica o nome do blog, a descrição e o botão de tema. É como a capa de um caderno.

Títulos. h1 é o maior (nome do blog), h2 é menor (título do post). Quanto menor o número, maior e mais importante o título.
Diz pro navegador que aquele bloco é um conteúdo independente — no caso, um post do blog. Mostra uma imagem. O src é o endereço da foto. O alt é um texto que aparece se a imagem não carregar — e ajuda quem usa leitor de tela.
